# Структура базы знаний по работе
## Наименование разделов и их содержание
- папки группируют объекты знаний, именуются на английском языке Title Case 
- файлы именуются объектом знания на английском языке CamelCase, Kebab-Case, Title Case
- заголовки внутри файлов именуют мем объекта на русском языке с заглавной буквы

## Структура
- Conventions - соглашения и стандарты
- Languages - синтаксис языков программирования, разметки, запросов
- Processes - процессы (методологии, регламенты, инструкции)
- Software - используемое ПО (приложения, утилиты, ОС)
