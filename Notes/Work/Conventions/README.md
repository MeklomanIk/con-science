# README
- файл, хранящий  информацию о репозитории
## На какие вопросы должен отвечать README?
- Что это за проект? (краткое назначение)
- Для кого он предназначен? (целевые пользователи)
- Какие основные возможности/цели? (что делает)
- Каков текущий статус / maturity? (alpha/beta/production)
- Какие требования и зависимости? (OS, инструменты, версия - языка)
- Как быстро запустить проект локально? (quick start)
- Как открыть dev container / среду разработки?
- Как устроена структура репозитория? (ключевые директории/- файлы)
- Как выполнять тесты и линтинг?
- Как вносить вклад / процесс pull request?
- Куда сообщать об ошибках и запросах фич? (issues, контакты)
- Лицензия и права использования
- Планы и roadmap (кратко)
